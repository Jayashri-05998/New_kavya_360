## 🔧 Configuration Quick Reference

### Backend Environment Variables (Must Set)
```
SPRING_DATASOURCE_URL=jdbc:mysql://${{ MySQL.MYSQL_HOST }}:${{ MySQL.MYSQL_PORT }}/${{ MySQL.MYSQL_DATABASE }}
SPRING_DATASOURCE_USERNAME=${{ MySQL.MYSQL_USER }}
SPRING_DATASOURCE_PASSWORD=${{ MySQL.MYSQL_PASSWORD }}
PORT=8080
SPRING_PROFILES_ACTIVE=prod
SENDGRID_API_KEY=<your_key>
EMAIL_USER=<your_email>
EMAIL_PASS=<your_password>
```

### Frontend Environment Variables (Must Set)
```
VITE_API_BASE_URL=https://<backend-domain>/api
VITE_API_URL=https://<backend-domain>
NODE_ENV=production
```

### Where to Get Values
- **MySQL credentials**: Auto-generated by Railway
- **SendGrid API key**: https://sendgrid.com
- **Gmail app password**: https://myaccount.google.com/apppasswords
- **Backend domain**: Railway provides after deployment
- **Frontend domain**: Railway provides after deployment
